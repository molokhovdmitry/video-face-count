About

This web application scans video frames for faces with specified periods of time and makes a plot with that data.

GitHub

How To Use

1. Upload a video file with .mp4 or .avi file format.

Higher resolution videos are processed slower.

2. Enter a scan period with which a video frame is scanned.

Application always starts with scanning the first frame.

3. Press the "Make a Plot" button to start.

Processing can take a while.

Privacy

Any file you upload/process is deleted from the server instantly after plot completion, but the following information is saved on the database: filename, extension, duration, frames per second, dimensions, scan period of the video file.
